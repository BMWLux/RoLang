Follow these simple steps to install and run RoLang from your terminal:

1. Clone the Repository
```bash
git clone https://github.com/BMWLux/RoLang.git
cd RoLang
```

2. Install RoLang
Make sure you have Python 3 installed, then run:
```bash
pip install .
```

3. Run RoLang in Terminal
After installation, use this command to start the RoLang CLI:
```bash
rolang
```

Or, if needed:
    bash
python -m rolang.cli
